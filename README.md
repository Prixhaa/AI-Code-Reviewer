# 📌 Overview

AI Code Reviewer is a lightweight static code analysis tool built using Python. It analyzes Python source files and generates insights into code quality, structure, and maintainability.

The project is designed as a foundation for an AI-assisted code review system that can be extended with Large Language Models (LLMs) for intelligent feedback and automated code improvement.

---

# 🚀 Features

* Analyze Python source files
* Count functions and classes
* Detect TODO comments
* Count blank lines
* Count comment lines
* Generate basic code quality suggestions
* Terminal-based review report
